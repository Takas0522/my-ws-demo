# 1.アプリケーションを動かす

## 目的

とにもかくにもアプリケーションを動かせないと始まりません。アプリケーションのデバッグ実行の仕方を特定する必要があります。

## 手段

GitHub Copilotを使用してアプリケーションの実行方法を特定します。特定された実行方法をドキュメント化すれば後任の方々のためになります。

``` md
アプリケーションのデバッグ実行の方法を特定しルートディレクトリのREADME.mdに追記してください。
Bffのコードを参照し、各サービスで必要なポート番号を指定するようにしてください。
各サービスのコマンドを記述してください。簡易化の為ルートディレクトリからの移動も含めた状態で構成してください。
```
