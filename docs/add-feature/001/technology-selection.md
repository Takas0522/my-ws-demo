# 技術選定結果

## 使用技術
1. **Frontend**:
   - Vue.js: コンポーネントベースのUI構築。
   - Axios: API通信ライブラリ。
2. **Backend**:
   - Payara: Java EE ベースのマイクロサービスアーキテクチャ。
   - JWT: ユーザー認証。
3. **Database**:
   - PostgreSQL: データ永続化。
4. **API**:
   - RESTful API: サービス間通信。

## 選定理由
- **Vue.js**: 軽量で学習コストが低く、SPA構築に適している。
- **Payara**: Java EE ベースで、既存サービスとの統一性を確保。
- **PostgreSQL**: 高い信頼性とスケーラビリティを持ち、既存サービスとの統一性を確保。
