# システム全体構成

以下は、ポイント管理機能を含むシステム全体の構成です。

## コンポーネント
1. **Frontend**:
   - ユーザーインターフェースを提供。
   - ポイント残高や履歴を表示。
2. **BFF (Backend for Frontend)**:
   - フロントエンドとバックエンド間の通信を仲介。
   - APIの集約とデータ整形を担当。
3. **Point Service**:
   - ポイント残高や履歴を管理。
   - データベース操作を担当。
4. **Auth Service**:
   - ユーザー認証を担当。
   - JWTトークンの検証を行う。
5. **User Service**:
   - ユーザー情報を管理。
   - 他サービスにユーザー情報を提供。
